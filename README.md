# XOX-AI

A minimax based XOX ai.</br>
You can learn the minimax algorithm from here https://en.wikipedia.org/wiki/Minimax
</br>

# How to use:

- Download the source code
- Download wxWidgets 
- Change wxWidgets path in the project linker and compiler settings
- Build
